# Release Notes Template

## 使用方法

将 AI 生成的更新内容替换下面的模板内容，然后保存为文件（如 `release-notes-v1.0.1.md`），使用 `-f` 参数创建 Release：

```bash
./create-release.sh -t v1.0.1 -f release-notes-v1.0.1.md
```
